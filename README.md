# RevolutionVA Pop-up Meetups

Pop-up Meetups is a RevolutionVA community initiative that brings developer talks to neighborhoods, coffee shops, libraries, coworking spaces, and other informal venues across Hampton Roads. If you have an idea worth sharing, we want to help you share it.

## How It Works

Anyone in the Hampton Roads tech community can propose a talk. Proposals go through a lightweight review process managed entirely through GitHub Issues. Once approved, RevolutionVA will *help* speakers coordinate their events and list it on [757tech.org](https://757tech.org) and the [757 Developers Meetup group](https://www.meetup.com/757dev/).

### For Speakers

1. [Open a new issue](../../issues/new/choose) using the **Talk Proposal** template.
2. Fill out the details — title, abstract, format, preferred location, preferred date, and any AV needs.
3. The review team will respond within 7 days. We may ask follow-up questions or suggest adjustments.
4. Once approved, we'll work with you to schedule and announce the event.

### For Venues

Want to host a Pop-up Meetup? Open an issue using the **Venue Submission** template. We're looking for spaces across all 757 cities — Norfolk, Virginia Beach, Chesapeake, Hampton, Newport News, Suffolk, and Williamsburg.

## Review Process

Proposals are reviewed by the RevolutionVA team. We evaluate submissions based on:

- **Relevance** — Does the topic serve the Hampton Roads developer community?
- **Clarity** — Is the abstract clear enough for attendees to know what they're signing up for?
- **Feasibility** — Can we reasonably support the talk format and logistics?

We aim to be inclusive and encourage first-time speakers. If your proposal needs work, we'll tell you how to improve it rather than just declining.

Labels used in the review process:

| Label | Meaning |
|---|---|
| `needs-review` | Newly submitted, awaiting triage |
| `in-review` | Actively being evaluated |
| `approved` | Approved and being scheduled |
| `scheduled` | Date and venue confirmed |
| `changes-requested` | Feedback sent, waiting on speaker |
| `declined` | Not a fit at this time |
| `venue-submission` | Venue proposal (not a talk) |

## Code of Conduct

All Pop-up Meetup participants — speakers, attendees, and hosts — are required to follow the [RevolutionVA Code of Conduct](CODE_OF_CONDUCT.md). To report an incident, contact us at team@revolutionva.org.
